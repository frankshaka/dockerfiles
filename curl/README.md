# Description

An *[Alpine-based](https://alpinelinux.org)* image with `curl` installed.

# How to use this image

To send a request,

```bash
$ docker run --rm -it frankshaka/curl curl -vvv "https://www.google.com"
```

To launch the shell,

```bash
$ docker run --rm -it frankshaka/curl sh
```
